## 常见问题
### 1. MP/SE/IC（BE）版本的区别
- 价格不同
参考官网的价格，IC（BE）<SE<MP。
- 功能与变量支持

>- IC：标准版，支持2048个变量，可满足日常需要；
>
>- SE：特别版，相比于IC，优势就是变量最大支持32767个变量；
>
>- MP：在SE基础上加入并行计算能力，支持最大120000个变量和超过20亿个观测值，支持多核cpu，能够加快运算速度。

所以，能下载使用MP版本，就不要下载BE/SE版本了！
